# team-members-list
FTS-TASK2 in HTML/CSS.

The task represents a list of team members. The program provides the capability of adding team members to a list and searching through them.

### Focus areas:
- Gradients.
- From custom styling.
- Accessible Forms.
- RWD (Responsive Web Design).
- Animations (optional).
